# 概要
シフト提出、編集、閲覧をスマホから簡単に行えるWebアプリです。  
http://shiftmanagementapp.com
# 使用技術
**フロントエンド**
- jQuery 3.4.1
- Bootstrap 4.3.1
- Googlechart
- FullCalender 5.10.2
- HTML/CSS
  
**バックエンド**
- Python 3.7.10
- Django 3.2.13
  
**インフラ**
- AWS
    - EC2,RDS,S3,SES,Route53
